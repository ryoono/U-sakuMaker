# 概要  
ゆうさく注意喚起シリーズを手軽に作成できるツールです  
後悔はしていない  
なお公開はする模様(激寒)  


# 使い方  
python3.xをインスコ   
(Python 3.9.7なら確実に動く)  

おぼろげながら浮かんできた単語を引数に実行する  
```sh
$ python3 yusaku.py [出力したい文字列]
# 引数には15文字以下の文字列を指定
```
「gif」ディレクトリに結果が格納される  
終わり！閉廷！…以上！皆解散！
